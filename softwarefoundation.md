# Mission : Atmosoft Foundation (AF)

These are the most prominent and influential software foundations, previously discussed, consolidated here for completeness.

1. **Apache Software Foundation (ASF)**  
   - **Overview**: Founded in 1999, supports a wide range of open-source projects.  
   - **Mission**: Provide software for the public good via community-driven development.  
   - **Key Projects**: Apache HTTP Server, Hadoop, Kafka, Spark.  
   - **Link**: [apache.org](https://www.apache.org/)

2. **Linux Foundation**  
   - **Overview**: Established in 2000, supports Linux and other open-source technologies.  
   - **Mission**: Accelerate open technology development and adoption.  
   - **Key Projects**: Linux Kernel, Kubernetes, Node.js.  
   - **Link**: [linuxfoundation.org](https://www.linuxfoundation.org/)

3. **Free Software Foundation (FSF)**  
   - **Overview**: Founded in 1985 by Richard Stallman, advocates for free software.  
   - **Mission**: Promote user freedom and software rights.  
   - **Key Projects**: GNU, GCC, GIMP.  
   - **Link**: [fsf.org](https://www.fsf.org/)

4. **Mozilla Foundation**  
   - **Overview**: Founded in 2003, focuses on an open, privacy-respecting internet.  
   - **Mission**: Keep the internet a global public resource.  
   - **Key Projects**: Firefox, Thunderbird, Rust.  
   - **Link**: [mozilla.org](https://www.mozilla.org/)

5. **Eclipse Foundation**  
   - **Overview**: Established in 2004, supports development tools and IoT.  
   - **Mission**: Provide a vendor-neutral platform for open-source innovation.  
   - **Key Projects**: Eclipse IDE, Jakarta EE, Eclipse Che.  
   - **Link**: [eclipse.org](https://www.eclipse.org/)

6. **Python Software Foundation (PSF)**  
   - **Overview**: Founded in 2001, supports the Python language ecosystem.  
   - **Mission**: Promote and advance Python.  
   - **Key Projects**: Python, PyPI.  
   - **Link**: [python.org](https://www.python.org/)

7. **Open Source Initiative (OSI)**  
   - **Overview**: Founded in 1998, defines and promotes open-source licensing.  
   - **Mission**: Educate about and advocate for open source.  
   - **Key Projects**: Supports Git, WordPress (via licensing).  
   - **Link**: [opensource.org](https://opensource.org/)

8. **Cloud Native Computing Foundation (CNCF)**  
   - **Overview**: Founded in 2015, promotes cloud-native technologies.  
   - **Mission**: Make cloud-native computing ubiquitous.  
   - **Key Projects**: Kubernetes, Prometheus, Envoy.  
   - **Link**: [cncf.io](https://www.cncf.io/)

9. **OpenSSF (Open Source Security Foundation)**  
   - **Overview**: Launched in 2020, focuses on open-source security.  
   - **Mission**: Secure critical open-source software.  
   - **Key Projects**: Sigstore, OpenSSF Scorecard.  
   - **Link**: [openssf.org](https://openssf.org/)
